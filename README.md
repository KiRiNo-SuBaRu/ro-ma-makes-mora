# ro-ma-makes-mora

**ro-ma-makes-mora** is an experimental way of writing Japanese using Roman letters.  
It plays with **capital letters** to show the “beats” inside Japanese words.

In Japanese, each kana (あ, か, きょ, ん, small っ, etc.) can be felt as one beat of rhythm,  
called a *mora* — like the claps you count in haiku or song lyrics.  
This project tries to make those hidden beats visible when you write Japanese in romaji.

By marking each beat (mora) with capital letters, this style makes it easier to see at a glance whether a string of letters is:
    1. a word from a Western language, or  
    2. Japanese written in romanized form.

The goal is to help people who are curious about Japanese  
to “see” the rhythm of the language, and for creators to use Japanese‑looking text in a more playful, conscious way.

> **Important:**  
> This is a *style* of writing, similar to a calligraphic or handwritten style, not a new official spelling.  
> It does **not** change standard romanization rules.  
> Think of it as a visual trick added **on top of** normal romaji, only to highlight these rhythm beats (mora).
